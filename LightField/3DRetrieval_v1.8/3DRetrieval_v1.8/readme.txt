1. folder "3DAlignment_v1.8" is the source code for calculating feature for 3D models.
2. folder "LightField_v1.8" is the source code for generating Precise-Recall diagram.
3. folder "Executable" is executed files of both. There is a example with 4 3D models.
4. file "FileName_v1.19.exe" is a tool, it can record folder and file name into a file. When running the file, you can use '1.' to record. Use "XnameX" to denote the filename, use "XpathX" to denote path and use "XpathnameX" to denote path and file name. You can also record it in *.bat, and run it later.
